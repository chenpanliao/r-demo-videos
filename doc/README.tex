\section{宣言}

《R 演習室》系列影片（以下簡稱為本系列影片）是R語言在統計學上應用的教學影片，由Chen-Pan Liao於2012至2013年間製作，內容主要是利用R進行常見的統計分析或檢定，預設觀眾是最初階的R使用者。

除非另有聲明，本文件（《R 演習室》系列影片簡介）及本系列影片一律以\nbs{}\href{http://creativecommons.org/licenses/by-sa/3.0/}{Creative Commons Attribution-ShareAlike 3.0 Unported License}\nbs{}授權。\footnote{CC BY-SA 3.0網址為\nbs{}\url{http://creativecommons.org/licenses/by-sa/3.0/}\label{ccbysa}}歡迎在該授權的規範下重製、散布、傳輸、修改或販售這些影片。

本系列影片之作者並不保證所有影片內容的正確性，請勿完全依賴影片內容以進行專業工作。
